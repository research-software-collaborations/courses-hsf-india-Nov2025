## Local setup

First, clone this repository and checkout the `python` branch.

```shell
git clone https://github.com/research-software-collaborations/courses-hsf-india-Nov2025.git

cd courses-hsf-india-Nov2025

git checkout python
```

Make sure you've [installed pixi](https://pixi.sh/latest/installation/) on your computer.

Then you can install the environment and start a local JupyterLab session with:

```shell
pixi run start
```
